สัญญาณเตือนเพื่อความปลอดภัย
Alarm Security

ระบบดูแลด้านความปลอดภัยด้วยเทศโนโลยีระบบอัติโนมัติที่เชื่อมต่อกับบอร์ตอดูโน่ โดยรับสำสั่งมาจาก Codeที่เรา
ป้อนเข้าไป ซึ่งระบบหลักๆที่ใช้ตรวจจับด้านความปลอดภัยมี การตรวจจับเปลวไฟและการตรวจจับ เมื่อวัตถุเข้ามาใน
ระยะที่กำหนดไว้ อีกทั้งระบบได้ติดตั้งไว้บนแกนหมุนมอเตอร์ที่คอยหมุนตรวจจับความปลอดภัย อยู่ที่ 180 องศา
และระบบจะแสดงผลออกมาในรูปแบบการแจ้งเตือนภัยแบบเสียง แสงไฟ และจอแสดงผล

Infomation

Feature

- Alarm Security สามารถหมุนตรวจจับความปลอดภัยได้ 180 องศา

- ตรวจับเปลวไฟและวัตถุที่เข้ามาในระยะอันตรายได้ตามที่กำหนด

- แสดงผลการแจ้งเตือนความปลอดภัยได้ 3 รูปแบบคือ เสียง แสงไฟ และจอแสดงผล

- ในการแจ้งเตือน จะแสดงผลออกมาไม่เหมือนกัน เพื่อให้รู้ชนิดของการแจ้งเตือน

- การแจ้งเตือนเปลวไฟ จะแสดงผลออกมาในรูปแบบแสงไฟสีขาวกับเหลือง พร้อมมีเสียง
อีกทั้งจอแสดงผลจะขึ้นคำว่า SOS และ HELP สลับไปมา

- ถ้าไม่มีเปลวไฟหรือวัตถุที่เข้ามาในระยะอันตรายที่กำหนดไว้ มอเตอร์ก็จะหมุนตรวจจับ
ไป-กลับเรื่อยๆ โดยไม่มีการแจ้งเตือน
